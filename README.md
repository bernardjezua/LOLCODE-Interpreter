# LOLCODE Interpreter Python App
## Installation

* to install the tkinter dependency

```
pip install tk
```

## Run the code

* Running the code without the GUI. Do note that it will not run the GIMMEH correctly.
```
python main.py
```

* Running the code with the GUI.
```
python .\frontend\main.py
```
